SQL Basics

16 August 2017 SQL, NoSQL
Basic Concepts Database: A container (usually a file or a set of files) to store organized data. Table: A structured list of data of a specific type. Every table in the database has a unique name that identifies it. Schema: Information about database and table layout and properties. For example, information that how data is stored in them, what data is stored , how it’s broken up etc. Primary Keys: A column(or a set of columns) whose value uniquely identify every row in a table.
